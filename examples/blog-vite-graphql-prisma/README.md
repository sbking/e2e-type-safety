# Blog Example - Vite + GraphQL + Prisma

This directory contains an example blogging application using React with Vite, an Apollo GraphQL server, and a PostgreSQL database with Prisma.

## Getting Started

```sh
cd server
cp .env.example .env
pnpm install
pnpm run docker
pnpm run setup
pnpm run dev
```

```sh
cd client
cp .env.example .env
pnpm install
pnpm run dev
```
